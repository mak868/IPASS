YL-40 ADC/DAC chip.

Author
Thomas Theil
Date
26-06-2018 14:00

This class controles the YL-40 board using the PCF8591 chip.
The PCF891 chip is a A/D and D/A converter controled via I2C.
for the data sheet folow This link. After boot the chip wil auto read Analog in 0

i2c adress
the i2c adress is 0x48.
Precondition
This chips needs a hwlib i2c to work

YL-40 initialize(exampel)

auto scl = target::pin_oc( target::pins::scl );
auto sda = target::pin_oc( target::pins::sda );
auto i2c_bus = hwlib::i2c_bus_bit_banged_scl_sda( scl, sda );
YL40 ADCi2(i2c_bus, 0x48);
Warning
This class needs hwlib to work.
