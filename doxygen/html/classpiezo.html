piezo::piezo ( hwlib::target::pin_out &  output)

Piezo buzzer class.

Author
Thomas Theil
Date
25-06-2018 15:30

This class wil output a sound over a piezo buzzer.

Precondition
To construct this class will need a output pin.

Piezo initialize

auto buzzerPin = target::pin_out(target::pins::d5);
piezo buzzer(buzzerPin);
Warning
This class requers hwlib to work.
